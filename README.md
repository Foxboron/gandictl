gandictl
========

Basic livedns client for Gandi.

```
$ GANDICTL=$(pass api/gandi) gandictl             
Edit gandi stuff

Usage:
  gandictl [command]

Available Commands:
  edit        edit
  help        Help about any command
  list        list
  records     records

Flags:
  -h, --help   help for gandictl

Use "gandictl [command] --help" for more information about a command.
```
